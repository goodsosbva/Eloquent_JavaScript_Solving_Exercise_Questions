# 콘웨이의 인생 게임 (Conway's Game of Life)

## 프로젝트 소개

콘웨이의 인생 게임(Conway's Game of Life)을 구현한 웹 애플리케이션입니다. 이는 세포 자동자(cellular automaton)의 한 종류로, 간단한 규칙에 따라 인공 생명체가 진화하는 시뮬레이션입니다.

## 주요 기능

- **30x30 그리드**: 체크박스로 구성된 인터랙티브 그리드
- **랜덤 초기화**: `Math.random()`을 사용한 랜덤 패턴 생성
- **자동 재생**: 3초마다 자동으로 다음 세대로 진행
- **수동 제어**: 체크박스를 클릭하여 패턴을 직접 편집 가능
- **세대 카운트**: 현재 세대 수를 실시간으로 표시

## 게임 규칙

1. **살아있는 셀 (Living Cell)**

   - 이웃이 2개 미만이면 → 죽음 (과소밀집)
   - 이웃이 2-3개면 → 생존
   - 이웃이 3개 초과면 → 죽음 (과밀집)

2. **죽은 셀 (Dead Cell)**

   - 이웃이 정확히 3개면 → 부활 (번식)

3. **이웃의 정의**

   - 8방향 (상하좌우 + 대각선)의 인접한 셀

4. **규칙 적용 원칙**
   - 모든 셀에 동시에 규칙이 적용됨
   - 현재 세대의 상태를 기준으로 다음 세대가 계산됨

## 사용 방법

1. `index.html` 파일을 브라우저에서 엽니다
2. 페이지 로드 시 자동으로 랜덤 패턴이 생성되고 자동 재생이 시작됩니다
3. **다음 세대** 버튼: 수동으로 한 세대 진행
4. **자동 재생 시작/중지** 버튼: 자동 재생 제어
5. **초기화** 버튼: 새로운 랜덤 패턴으로 재시작
6. 체크박스를 클릭하여 패턴을 직접 편집할 수 있습니다
